# 公告

本项目涉及侵权行为的违法，也为此承担了相应的法律责任，在此郑重发布公告，删除项目内容并规劝所有人：

请立刻停止一切侵权相关行为，不要再效仿、从事、传播各类侵权活动，切勿触碰法律红线，不要抱有任何侥幸心理。

法律底线不可逾越，违法违规必将付出代价，望大家恪守法律规定，自觉遵纪守法，远离所有侵权行为。
